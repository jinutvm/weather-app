Welcome to my Example HTML Document

This is a sample HTML document. It includes some basic HTML tags, such as:

  • <html> and </html>: the opening and closing tags for the entire document.
  • <head> and </head>: the opening and closing tags for the document's head section, which contains metadata and other information about the document.
  • <body> and </body>: the opening and closing tags for the document's body section, which contains the visible content of the document.
  • <title> and </title>: the opening and closing tags for the document's title, which appears in the browser's title bar.
  • <meta>: a tag used to provide metadata about the document, such as the character set and viewport settings.
  • <style> and </style>: the opening and closing tags for inline CSS styles that can be applied to the document's elements.
  • <h1> and </h1>: the opening and closing tags for a top-level heading.
  • <p> and </p>: the opening and closing tags for a paragraph of text.
  • <ul> and </ul>: the opening and closing tags for an unordered list.
  • <li> and </li>: the opening and closing tags for a list item.

Feel free to modify this document and experiment with different HTML tags and styles!
